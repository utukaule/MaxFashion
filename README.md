# Max-Fashion
The Max Fashion website is an online web application that caters to clothes for all genders.




## Tech Stack
- HTML
- CSS
- BootStrap
- JavaScript
- Local Storage





## Features

- Sign-in/Sign-up Page
- Navigation Bar
- Filtering Item by price and categories
- Add to Wish List
- Add to Cart
- Delete from Cart
- Checkout Page
